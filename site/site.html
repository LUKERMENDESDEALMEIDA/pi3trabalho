O Trabalho

A Proposta do trabalho era fazer um bot do discord para ler e reconhecer imagens de um dataset pré-disposto aos alunos.

Libraries

As Libs utilizadas foram:
Tensorflow
keras
Pandas
Pillow
numpy
Scikit


Imagem do Bot Funcionando


Corpo

O bot foi feito inteiramente em Python utilizando o data-set de Placas de transito alemãs disponibilizado pelo professor
O Epoch total usado foi de 50 vezes, o bot conta com uma precisão de 96% de acertos.

O Keras foi usado como cérebro ele é o que calcula e aprende a discernir as imagens
o tensorflow foi implementado como uma espécie de telemetria para sabermos como anda o aprendizado
O Pillow para a manipulação da imagems junto com o Numpy para a transformação em vetor
e o Scikit como suplemento ao Keras
